# "어린이 수학" 비공개 테스트 안내

테스트에 참여해주셔서 정말 감사합니다 🙏
14일 동안 잠깐만 도와주시면, 어린이들이 즐겁게 수학을 배울 수 있도록 큰 힘이 됩니다.

---

## 📥 1. 설치 방법

### 단계별

1. 아래 링크를 **안드로이드 폰**에서 열어주세요:
   ```
   https://play.google.com/apps/testing/com.rts.rys.ryy.math.children_math_game
   ```

2. **"테스터 되기"** (또는 "Become a tester") 버튼을 누르세요.

3. 같은 화면에서 **"Google Play에서 다운로드"** 링크를 누르면 Play Store가 열립니다.

4. 평소 앱 설치하시듯이 **"설치"** 버튼을 누르면 끝!

### ⚠️ 설치가 안 되면

- "App not available" 메시지가 보이면 **30분~1시간 후 다시 시도**해주세요 (Google 서버 전파 시간).
- 그래도 안 되면 안내 받으신 이메일 계정이 폰의 Google Play에 로그인된 그 계정과 같은지 확인 부탁드려요.
- 계정 확인 방법: Play Store 앱 → 우상단 프로필 아이콘 → 로그인된 계정 표시됨

## 🎮 2. 14일간 부탁드리는 것 (정말 잠깐이에요)

Google이 비공개 테스트를 "활성"으로 인정하려면 단순 설치가 아닌 **실제 사용**이 필요합니다.

### 매일 (꼭 부탁드려요)

- ⏰ **하루 한 번**, 앱을 켜주세요
- 🎯 **문제 1~2판 풀기** (어떤 모드든, 어떤 연산이든 OK)
- ⏱️ 소요 시간: **30초 ~ 3분**

> 14일 × 3분 = 누적 42분. 출퇴근 길 잠깐 켜주시면 됩니다.

### 한 번씩만 (있으면 정말 좋아요)

테스트 기간 중 **딱 한 번씩만**이라도 시도해주시면 다양한 화면을 거치면서 숨은 버그를 찾는 데 도움이 됩니다:

- [ ] 첫 실행 시 튜토리얼 5장 모두 넘겨보기
- [ ] 이름 변경 (홈 상단 ✏️ 아이콘 탭)
- [ ] **4가지 연산 각각** 한 판씩 (덧셈 / 뺄셈 / 곱셈 / 나눗셈)
- [ ] **3가지 모드 각각** 한 판씩 (도전 / 연습 / 타임어택)
- [ ] **구구단 모드** 한 판
- [ ] **혼합 모드** 한 판
- [ ] 도장판 진입 → 받은 도장·배지 구경
- [ ] 통계 화면 진입 → "약점 분석" 그리드 보기
- [ ] 음소거 토글 한 번 (홈 우상단 스피커 아이콘)
- [ ] 앱 종료 후 재실행 — 튜토리얼이 다시 안 뜨는지 확인

## 🐛 4. 문제·의견 보내는 법

테스트 중 발견하신 어떤 것이든 답장 부탁드려요. 작은 거라도 큰 도움이 됩니다.

### 알려주시면 좋은 것

- **버그**: 앱이 멈추거나 이상하게 작동하는 화면
- **헷갈리는 부분**: "이게 뭐 하는 버튼인지 모르겠다"
- **수학 문제 자체 이슈**: 답이 이상하게 나오거나 너무 어려운 문제
- **자녀의 반응**: 아이가 직접 해보셨다면 어떤 점을 좋아했는지/싫어했는지

### 알려주시는 방법

- 답장 메시지 (카톡 / 문자 / 메일 — 받으신 채널 그대로)
- 가능하면 **스크린샷** 함께 보내주시면 더 좋아요 (화면 잘못된 부분 빨리 파악됨)
